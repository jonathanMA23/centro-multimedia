# Centro Multimedia

Una aplicación de escritorio construida con Python que permite gestionar y automatizar tareas multimedia.

## Funcionalidades
* **Interfaz Gráfica (GUI):** Construida con `tkinter` para un uso sencillo.
* **Automatización:** Uso de `pyautogui` para controlar el ratón/teclado y `subprocess` para ejecutar comandos del sistema.

## Requisitos Previos

Para ejecutar este proyecto necesitas tener instalado:
* [Python 3.x](https://www.python.org/downloads/)
* Pip (Gestor de paquetes de Python)

## Instalación

1. Clona el repositorio:
   ```bash
        git clone [https://github.com/jonathanMA23/centro-multimedia](https://github.com/jonathanMA23/centro-multimedia)
   ```

2. Navega a la carpeta del proyecto:
* cd centro-multimedia

3. Instala las bibliotecas necesarias:
```bash
    pip install pyautogui
```
4. Ejecuta el archivo
```bash
    python multimedia.py
```
